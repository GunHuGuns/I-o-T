🚀 快速开始
─────────────────────────────────────────────────────────────────────────────

1️⃣  解压文件
   tar -xzf project-html.tar.gz
   cd project-html

2️⃣  启动服务器（选择一种）
   
   选项 A - 自动启动（推荐）
   - Linux/Mac:  ./START-HERE.sh
   - Windows:    START-HERE.bat
   
   选项 B - 手动启动
   - Python 3:   python -m http.server 8000
   - Python 2:   python -m SimpleHTTPServer 8000
   - Node.js:    npx http-server -p 8000

3️⃣  打开浏览器
   访问：http://localhost:8000

❓ 常见问题
─────────────────────────────────────────────────────────────────────────────

Q: 可以直接打开 HTML 文件吗？
A: 可以，但推荐使用本地服务器以获得最佳体验

Q: 需要安装什么？
A: 只需要浏览器。Python 或 Node.js 是可选的（用于启动服务器）

Q: 数据会保存吗？
A: 数据保存在浏览器 LocalStorage 中，清除浏览器数据会丢失

Q: 支持手机访问吗？
A: 支持，可通过局域网 IP 在手机上访问
   1. 查询计算机 IP: ipconfig (Windows) / ifconfig (Mac/Linux)
   2. 手机访问: http://your-ip:8000
